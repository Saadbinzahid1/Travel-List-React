# 🧳 Travel List – React SPA

A single-page React application that helps users organize and track items for their trips.

Users can add items, mark them as packed, view packing progress, and sort the list based on different criteria.

---

## 🌐 Live Preview

[Live Demo](https://travellistinreact.netlify.app)

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- JSX
- CSS
- Functional Components
- React State Management

---

## ✨ Features

- Add new travel items
- Mark items as packed/unpacked
- Delete items
- Dynamic packing progress counter
- Sort items by different parameters
- Responsive UI
- Single Page Application (SPA)

---

## 🎯 Purpose of This Project

This project was built to strengthen my understanding of React fundamentals, particularly state management and dynamic UI rendering in single-page applications.

It is part of my journey toward becoming a full-stack developer.
